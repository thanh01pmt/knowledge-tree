# 🏛️ Reference: Architecture & Pedagogical Foundations

Thư mục này chứa tài liệu nghiên cứu sư phạm và kiến trúc gốc của nền tảng LearnWell:

- **`Kiến Trúc và Lộ Trình Phát Triển Hệ Thống LearnWell.md`**: Tài liệu thiết kế tổng thể về kiến trúc Adaptive Learning, mô hình phân tách giữa Knowledge Tree và Curriculum, cùng lộ trình phát triển.
- **`Tính phù hợp và phổ quát của mô hình phân loại kiến thức học tập.md`**: Phân tích chuyên sâu về tính đúng đắn sư phạm của mô hình 6 tầng, nguyên lý Marr (Computational - Algorithmic - Implementational), và định hướng áp dụng AI / Graph Embeddings để suy luận cạnh `prerequisite`.
